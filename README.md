# Interactive Developer Portfolio

A developer portfolio designed as an interactive, IDE-like experience. This project simulates a code editor environment complete with a sidebar, status bar, an active code editor area, a functional terminal, and an immersive matrix background effect.

## Features

- 🖥️ **IDE-like Interface**: Features a familiar code editor layout with a sidebar, status bar, and main editor space.
- ⌨️ **Interactive Terminal**: An integrated terminal experience right in the browser.
- 🌌 **Matrix Background**: A sleek, animated matrix digital rain background.
- ⚡ **Modern Stack**: Built using React 19, TypeScript, and Vite for lightning-fast performance.
- 🎨 **Styling & Icons**: Styled using Tailwind CSS v4 and standard Lucide React icons.

## Getting Started

### Prerequisites

- Node.js installed on your machine.

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Navigate into the project directory:
   ```bash
   cd portfolio
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Running the App

Start the development server:

```bash
npm run dev
```

Open your browser and navigate to `http://localhost:5173` to view the portfolio.

## Scripts

- `npm run dev` - Starts the Vite development server.
- `npm run build` - Compiles TypeScript and builds the application for production.
- `npm run lint` - Runs ESLint to statically analyze the code.
- `npm run preview` - Previews the production build locally.
